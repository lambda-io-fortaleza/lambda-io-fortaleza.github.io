Sobre

A Fundação Lambda I/O consiste de uma iniciativa aberta que visa introduzir as pessoas ao redor do mundo a todos os recursos e benefícios de se conhecer o paradigma de programação funcional. Nossa missão é desconstruir a doutrinação acerca do ensino de linguagens imperativas que, infelizmente, assola muitos lugares do mundo e, consequentemente, seus sistemas de ensino de TI - desde cursos técnicos a pós-graduações - fornecendo recursos e uma comunidade que tornaria muito mais fácil de se aprender sobre programação funcional, linguagens funcionais, casos de uso etc.

Credenciamento e Apresentação do Evento

9:30

Porque escolher o Elixir como sua próxima linguagem de programação

Marlus Saraiva

A evolução tecnológica que vem ocorrendo nos últimos anos vem trazendo desafios cada vez mais complexos para os desenvolvedores de softwares. Estamos saindo rapidamente de um modelo single-threaded, monolítico e centralizado para um modelo multi-threaded de microserviços altamente distribuídos e concorrentes. Nesta palestra vou mostrar porque o Elixir e a máquina virtual do Erlang são ideais para este novo cenário e como estas duas tecnologias podem ajudá-lo a produzir softwares de alta performance e escaláveis sem abrir mão da produtividade.

10:20

Pausa pra Lanche & Networking

10:40

Phoenix Framework: Do Começo a Produção

Schwarzenegger Alves

Phoenix é um framework Elixir para o desenvolvimento de aplicações web escaláveis. Juntos, vamos explorar ele, indo desde o básico a produção. Você verá princípios deste e como usar ele para escrever poderosas aplicações web.

11:30

Praticando Funcional com F#

Abraão Alves

Nessa palestra, serão mostradas técnicas de programação funcional com F# – e como isso pode levar o seu sistema a ter código mais limpo, simples e com menos bugs. Para demonstrar esse ambiente criamos uma aplicação web com direito a builds com o FAKE e instalação pacotes com o Paket.
